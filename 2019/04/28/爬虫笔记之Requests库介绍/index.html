定义

Requests 是用Python语言编写，基于 urllib，采⽤Apache2 Licensed 开源协议的 HTTP 库。
它比 urllib 更加方便，可以节约我们大量的工作，完全满足HTTP 测试需求。
一句话——Python实现的简单易用的HTTP库
